## 시리즈 클래스

데이터의 의미를 표시하는 index를 붙일 수있으며, 데이터 자체는 value라고 한다.

(Series class 는 넘파이에서 제공하는 1차원 배열과 비슷)

> **Series = value + index**

## 시리즈 인덱싱

시리즈는 넘파이 배열에서 가능한 인덱스 방법 외

인덱스 라벨을 이용한 인덱싱과 배열 인덱싱

인덱스 라벨을 이용한 slicing 슬라이싱 가능



시리즈 데이터를 인덱싱하면 값이 나온다.

